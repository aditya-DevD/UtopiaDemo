# Utopia Demo

A modern React-based web application built to demonstrate responsive UI design, component-based architecture, and clean frontend development practices.

## 🚀 Tech Stack

* **React.js** – Component-based UI development
* **JavaScript (ES6+)** – Application logic
* **HTML5 & CSS3** – Structure and styling
* **Git & GitHub** – Version control and collaboration

## 📌 Features

* Responsive and modern UI
* Reusable React components
* Clean and organized project structure
* Easy integration with APIs
* Scalable frontend architecture

## 🛠 Installation

Clone the repository:

```
git clone https://github.com/aditya-DevD/UtopiaDemo.git
```

Navigate to the project directory:

```
cd UtopiaDemo
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

or if using Create React App:

```
npm start
```
